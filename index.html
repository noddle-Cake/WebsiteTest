Housing Price Predictor

This project predicts housing prices using the Housing Price Index (HPI) data combined with Kaggle's location/price data.

Concepts Used:

  • Housing Price Index (HPI) data from FHFA
  • Price/Location data from Kaggle
  • Zip code location determination from city data
  • Machine Learning using Random Forests
  • Data Preprocessing with Python
  • Data Visualization using Plotly

Technologies Used:

  • NumPy
  • Seaborn
  • Matplotlib
  • Pandas
  • Jupyter Notebook
  • Python
  • TensorFlow
  • Plotly
